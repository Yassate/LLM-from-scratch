# GPT-2 Implementation From Scratch
This repository contains a step-by-step implementation of a GPT-2 style language model built from scratch using PyTorch. The project is designed as a hands-on learning exercise to better understand the internal mechanics of transformer-based large language models.

The implementation follows the excellent educational materials created by Sebastian Raschka:
- YouTube Series: https://www.youtube.com/playlist?list=PLQRyiBCWmqp5twpd8Izmaxu5XRkxd5yC-
- Reference Repository: https://github.com/rasbt/LLMs-from-scratch

## Purpose & Objectives
The focus of this repository is educational rather than performance-oriented. Instead of relying on high-level frameworks, most components are implemented directly to provide a deeper understanding of how modern LLMs operate internally.
The main goal of this project is to develop a practical understanding of:
- Transformer architecture
- Self-attention mechanisms
- Tokenization strategies
- GPT-style autoregressive training
- Text generation with language models

## Technologies Used
- **Python**
- **Jupyter Notebooks**
- **PyTorch**
- **tiktoken**

While `tiktoken` is used for GPT-compatible tokenization, a **basic tokenizer is also implemented manually** in order to explore and understand how tokenization works internally.
